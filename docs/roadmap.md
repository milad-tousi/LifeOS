# Roadmap

## Near term

- Add create, update, and archive flows for core domains
- Introduce input validation and form handling per feature
- Add view-level tests and repository tests

## Mid term

- Add Capacitor for Android and iOS packaging
- Bridge notifications and backup flows to native capabilities where beneficial
- Add richer dashboards and local analytics

## Long term

- Evaluate optional sync without breaking local-first behavior
- Add data compression and archival strategies for long-running datasets
- Expand domain depth while preserving a lightweight footprint
